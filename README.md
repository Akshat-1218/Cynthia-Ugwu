# Cynthia Ugwu Clone

A responsive front-end clone of Cynthia Ugwu's award-winning portfolio website, built using HTML, CSS, JavaScript, GSAP, and Locomotive Scroll.

## 🚀 Features

* Smooth scrolling using Locomotive Scroll
* GSAP page load animations
* Custom animated cursor
* Interactive image hover effects
* Responsive design for desktop and mobile devices
* Modern UI inspired by Cynthia Ugwu's portfolio

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP (GreenSock Animation Platform)
* Locomotive Scroll
* Remix Icons

## 🎯 Learning Outcomes

Through this project I learned:

* Advanced CSS layouts
* Responsive web design
* GSAP animations
* Smooth scrolling effects
* DOM manipulation with JavaScript
* Interactive UI development
